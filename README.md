# circleplot
Old mini-project for visualizing 2D matrices and their eigenvalues/-vectors. A line is drawn from each point on the unit circle to the point the matrix sends that point to.

For instance, here's [[-2, 1], [1, -1]] (a symmetric matrix with one EV being 0):
